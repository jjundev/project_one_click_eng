📋 AGENTS.md 업데이트 — 작업 회고
───────────────────────────────────────────

방금 완료한 작업을 회고하고 AGENTS.md, GEMINI.md를 업데이트하라.

## 1. 회고
아래 항목을 순서대로 스스로 점검한다.

- 이번 작업에서 실수, 오류, 예상과 다른 결과가 있었는가?
- 사용자의 수정 요청이나 피드백이 있었는가?
- 명령어를 잘못 실행하거나, 재실행이 필요했던 순간이 있었는가?
- 더 좋은 방법이 있었는데 돌아서 간 경우가 있었는가?

실수가 없었다면 → "이번 작업에서 기록할 실수 없음"을 한 줄 출력하고 종료.
실수가 있었다면 → 아래 2~4단계를 진행한다.

## 2. AGENTS.md에 실수 항목 추가
`## 🗂️ 실수 기록` 섹션 상단에 아래 형식으로 추가한다.

### [오늘 날짜] 실수 제목
- **심각도**: 🔴 / 🟠 / 🟡 / 🟢
- **카테고리**: 파일 조작 / 명령어 오류 / 경로 실수 / 의존성 / 기타
- **작업 컨텍스트**: (어떤 작업 중이었는지)
- **실수 내용**: (무엇을 잘못했는지)
- **원인 분석**: (왜 발생했는지)
- **실제 결과**: (어떤 영향이 있었는지)
- **올바른 방법**: (앞으로 어떻게 해야 하는지)
- **재발 방지 규칙**: `[ ] ...`

## 3. 금지 목록 및 체크리스트 갱신
- `## 🚫 절대 하지 말 것`에 재발 방지 규칙을 한 줄로 요약해 추가
- 필요하다면 `## ✅ 작업 전 체크리스트`에도 항목 추가

## 4. 통계 업데이트
- `## 📊 실수 통계` 해당 카테고리 카운트 +1
- 파일 하단 `마지막 업데이트` 날짜와 총 실수 건수 갱신
